# 🚀 ASP.NET Core MVC (.NET 10) + CoreUI Admin Template Integration

This repository is an ASP.NET Core MVC (.NET 10) application integrated with the **CoreUI Free Bootstrap Admin Template**, replacing the default MVC UI with a modern, responsive, and production-ready admin dashboard interface.

---

## 🎨 CoreUI Template

This project is based on the open-source CoreUI Admin Template:

- 🔗 Official website: https://coreui.io
- 🔗 GitHub repository: https://github.com/coreui/coreui-free-bootstrap-admin-template
- 🔗 Documentation: https://coreui.io/docs/
